# REST Resource Naming

## Overall Summary

Covers RESTful resource naming conventions and URI hierarchy best practices.

## Key Points

- Use plural nouns for collections (/users, not /user)
- Use hyphens for multi-word resources (/user-profiles)
- Nest resources to show relationships (/users/123/orders)
